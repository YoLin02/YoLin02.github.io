2025/6/9 技术笔记 教程

使用 Hexo 零基础搭建个人博客指南

一、搭建博客

1. 创建项目

mkdir myblog # 创建博客文件夹
cd myblog # 进入文件夹
npm install -g hexo-cli # 安装Hexo框架
hexo init # 初始化博客
npm install # 安装依赖

2.启动本地服务器

hexo clean && hexo g && hexo s

注意：需要在当前根目录myblog下

浏览器打开 http://localhost:4000 就能看到初始博客啦！
